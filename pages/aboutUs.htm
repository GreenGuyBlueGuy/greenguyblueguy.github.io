About Us

My games are inspired by conversations, usually while driving, with my kids.

Background

I have been teaching myself programming and art since I can remember. My first commercial release was in 2015, using Solar 2D to create a simple video tutorial mobile app.

To teach my kids how to make games, I started showing them RPG Maker and Roblox.

The Last Plant:

We joined the Global Game Jam and had a blast! The theme was "Roots," so we created The Last Plant.

Our kids made a lot of the artwork and sound effects, check it out here.

Roblox Games:

Roblox is a great introduction to game development for all ages. You will find Roblox provides an engine simple enough for kids and complex enough for a game studio.
